# Screenshots Directory

Place your app screenshots here for app store listings and PWA metadata.

## Required Screenshots
- `mobile-1.png` - Mobile view (750x1334 or similar portrait)
- `desktop-1.png` - Desktop view (1920x1080 or similar landscape)

## Tips
- Show your app's key features
- Use real data (or realistic mock data)
- Keep UI clean and professional
- Add more screenshots to showcase different features
